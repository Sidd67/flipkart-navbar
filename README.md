# Flipkart Navbar Clone

A pixel-perfect clone of the Flipkart navigation bar built with pure HTML and CSS.

![HTML5](https://img.shields.io/badge/HTML5-E34F26?style=for-the-badge&logo=html5&logoColor=white)
![CSS3](https://img.shields.io/badge/CSS3-1572B6?style=for-the-badge&logo=css3&logoColor=white)

## Live Demo

Open main.html in your browser to see the navbar in action.

## Preview

The project replicates Flipkart's iconic top navigation bar including:
* Search bar with autocomplete styling
* * Cart icon with badge
  * * Login/User section
    * * Fully responsive layout matching Flipkart's design system

      * Technology | Usage
